Oh Shit Wormhole is a freefalling game where you dodge obstacles while descending. For the controls, the mouse is used to move the player in various directions to dodge incoming obstacles. I will be using the gathered data to determine the game's user experience, ease of controls and the tutorial it has - e.g. how easy it was to realise the objective of the game and generally how it plays.
